# Portfólio - Ana Nascimento

Portfólio pessoal desenvolvido em React + Vite (SPA estático), hospedado na Vercel.

## Stack
- React 19 + Vite 7
- Tailwind CSS v4
- shadcn/ui

## Desenvolvimento
```bash
bun install
bun run dev
```

## Build
```bash
bun run build
```

Saída em `dist/`. Deploy automático via Vercel a cada push na branch principal.
